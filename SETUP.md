# 汚濁負荷量 共有アプリ セットアップ

複数人・別端末で同じデータを共有編集。担当者ごとのID/パスワード認証（退職者は個別に停止/削除＝即ログイン不可）。
データ用スプレッドシートは自動生成され、**GASを置いたGoogleアカウントのドライブ内**に作られます（他の下水アプリと同じ場所・アカウントで管理可能）。

## できること / できないこと（正直版）

- GitHub側（リポジトリ作成・ファイル配置・Pages公開）は **Claude Code で全自動**にできます（`CLAUDE_CODE.md`）。
- スプレッドシート作成・タブ作成・初期管理者作成は **GASが初回自動実行**。手動不要。
- ユーザーの追加/停止/削除/パスワード再設定は **アプリ内「ユーザー管理」**から（管理者ログイン時）。スプレッドシートを触る必要なし。
- **どうしても手動が残る部分**：GASの初回認可（GoogleのOAuth同意画面）と、ウェブアプリの「デプロイ→/exec URL発行」。
  これはGoogleが人手の同意を必須にしている仕様で、私（Claude）もClaude Codeも代行できません。所要5分・1回だけです。

## 手動で必要なのはここだけ（1回・約5分）

1. 対象Googleアカウント（他アプリと同じアカウント）で、適当なスプレッドシートを開き
   拡張機能 → Apps Script。既定コードを消し、同梱 `Code.gs` を貼り付け。
   （`appsscript.json` も使う場合は「プロジェクトの設定 → マニフェストを表示」で差し替え）
2. プロジェクトの設定 → スクリプト プロパティ に2つ:
   - `ODAKU_ADMIN_USER` = 初期管理者ID（例 `admin`）
   - `ODAKU_ADMIN_PASS` = 初期管理者パスワード（任意の強めの文字列）
   ※ `ODAKU_SHEET_ID` は未設定でOK（自動でシート生成し記録）。既存シートに入れたいなら、そのIDを設定。
3. デプロイ → 新しいデプロイ → ウェブアプリ。
   実行ユーザー=**自分** / アクセス=**全員（匿名含む）**。初回は認可（OAuth同意）を承認。
4. 発行された `…/exec` URL を `config.js` の `GAS_URL` に貼る。
   ```js
   window.ODAKU_CONFIG = { GAS_URL: "https://script.google.com/macros/s/XXXX/exec" };
   ```

これで完了。アプリを開く → 初期管理者でログイン → 「ユーザー管理」で各担当者を追加。

## 運用（退職者対応）

- アプリ上部「ユーザー管理」（管理者のみ表示）で:
  - **新規/更新**：ID・氏名・パスワード・権限を登録。
  - **停止**：そのユーザーのログインを即停止。既存セッションも無効化。
  - **削除**：完全削除（セッションも消去）。
  - **PW再設定**：パスワードのみ変更。
- 退職者が出たら「停止」または「削除」を押すだけ。データはサーバー（スプレッドシート）に残ります。

## 同時編集・オフライン

- 変更した日だけ送信。サーバーはLockServiceで直列化し**日単位マージ**（別々の日を同時編集→両方残る／同一日→後勝ち）。
- 約12秒ごとに他者の更新を受信。**自分が入力中（未送信）の日は上書きしない**。
- 一度開けばオフラインでも入力・Excel出力可（再接続時に同期）。●＝緑:同期済/黄:同期中/赤:オフライン。

## データ構造（自動生成タブ）

- `USERS` userId / name / salt / hash / active / role / …（パスワードはソルト付きSHA-256・1000回ストレッチで保存。平文は保持しない）
- `SESSIONS` token / userId / expiresAt（12時間）
- `DATA` key(年度__処理場__月) / fy / plant / month / json(31日配列) / rev / updatedAt / updatedBy
- `META` plant / json({code,name,writer}) / updatedAt / updatedBy

## 補足

- 出力ファイル（月報）は最初に渡された様式そのまま（値のみ差し込み・外部リンク除去）。サーバーは値の保管のみで書式は触らない。
- セキュリティ：エンドポイントは公開だが、認証はコード側（ユーザー表）で強制。限定運用前提。より強固にするなら将来 PBKDF2/bcrypt 相当へ差し替え可。
